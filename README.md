## 📚 Current Projects

### 🔐 Secure CI/CD Pipeline
* **Description:** An automated deployment pipeline engineered to integrate continuous security checks without delaying software delivery.
* **Tech Stack:** GitHub Actions / GitLab CI, SonarQube, Trivy, Docker.
* **Key Focus:** Static Application Security Testing (SAST), software composition analysis (SCA) for vulnerable dependencies, and container image vulnerability scanning.

### ☁️ Cloud Infrastructure Automation
* **Description:** A declarative Infrastructure as Code (IaC) setup designed to provision, configure, and harden cloud environments consistently.
* **Tech Stack:** Terraform, Ansible, AWS / Azure, Linux.
* **Key Focus:** Automated resource provisioning, configuration management, network segregation (VPCs), and enforcing system hardening baselines.

### 🛡️ Security Monitoring Dashboard
* **Description:** A centralized visual engine designed to aggregate system logs, track real-time operational vulnerabilities, and monitor security metrics.
* **Tech Stack:** Next.js, NestJS, Prometheus, Grafana.
* **Key Focus:** Real-time threat telemetry visualization, log aggregation parsing, and automated threshold alerts for anomalous behavior.

### 🔐 Secure Vault Manager
* **Description:** A security-focused credential and secret management application built to enforce zero-trust data sovereignty and cryptographic standards.
* **Tech Stack:** Pure Java, Java Cryptography Architecture (JCA).
* **Key Focus:** Data-at-rest encryption (AES-256), secure operational memory handling, and mitigation of local credential leaks.

### 🌍 PolyGlotExchange — Peer-to-Peer Language App
* **Description:** A mobile application matching native speakers looking to exchange and learn languages interactively (e.g., pairing an English speaker learning Spanish with a Spanish speaker learning English).
* **Tech Stack:** Flutter (Dart), NestJS (TypeScript), WebSockets.
* **Key Focus:** Native cross-platform UI engineering, dynamic matchmaking algorithms, and optimized real-time communication architecture.

### 🛰️ Low-Level TCP Socket Server-Client
* **Description:** A lightweight concurrent network messaging and file-transfer engine built from scratch to study transport-layer protocols and low-level mechanics.
* **Tech Stack:** Pure C, POSIX Threads, TCP/IP Sockets.
* **Key Focus:** Multi-threaded synchronization, memory safety, buffer management, and socket-level security hardening.

### 👁️ Smart Lighting System (SLS) Simulation
* **Description:** An automated edge-computing environment monitoring simulation designed to optimize municipal power grids using computer vision.
* **Tech Stack:** Python, YOLOv8, Pandas.
* **Key Focus:** Real-time object tracking pipelines, video feed stream optimization, and structured telemetry logging.
